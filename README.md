## Descrição
API REST para gestão de usuários e veículos, com autenticação JWT e controle de acesso por roles.

## Tecnologias
- ASP.NET Core
- Entity Framework Core
- MySQL
- JWT Authentication
- Swagger

## Funcionalidades
- Cadastro e login de usuários
- CRUD de veículos
- Controle de permissões
- Validações de regras de negócio

## Aprendizados
- Autenticação e autorização
- Arquitetura em camadas
- Boas práticas em APIs REST
